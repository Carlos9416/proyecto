Baja Usuarios

Descripcion:Con esta practica Aprendemos a eliminar los datos
de una base de datos tambien usando el lenguaje de
programacion JavaScript y el PHP.
Para poder dar de Baja a los usuarios, se necesita
que el usuario ingrese el nombre de usuario para que
el sistema busque al usuario en la base de datos con
un metodo creado con JS y PHP que validasi el usuario
existe dentro de la Base de Datos o no, en caso de
que si exista, se despliegan sus datos completos y ya se
puede dar de baja al usuario.
